# gradle-springboot-multi-project

## Intro
Example how to set up a multi-project sprinboot with gradle using non-default project paths

## Project layout
```bash
applications/
├── helloboot-service
│   └── src
│       ├── main
│       └── test
└── hellospring-service
    └── src
        ├── main
        └── test
```

## How-to
gradle clean build
